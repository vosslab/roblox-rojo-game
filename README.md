# Roblox Idle Tycoon Game

A kid‑friendly Roblox idle tycoon prototype with a simple quest loop, server‑authoritative economy, and a scripted playground built entirely from Parts.

## Quick start
1. Install tooling (Homebrew): see [`docs/INSTALL.md`](docs/INSTALL.md).
2. Start Rojo and connect in Studio:
   - `rojo serve`
   - In Roblox Studio: Plugins → Rojo → Connect
3. Run checks:
   - `tests/run_all_tests.sh`

## What’s inside
- Server‑side services for world building, economy, quests, and persistence.
- Client‑side UI and input controllers.
- Minimal physics interactables (swing, slide, merry‑go‑round).
